# qvblog
个人博客系统前端代码（基于react开发）
## 项目技术栈
1 react 
2 redux
3 react-router-dom
4 富文本插件
5 less
6 webpack 各种插件配置
## 安装依赖
```
npm install
```
## 运行代码
```
npm start
```
## 项目打包
```
npm build
```
